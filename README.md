## Technologies used. 🧪
<br/>

-  React JS
-  Javascript
-  Vite JS
-  Tailwind CSS

<br/>

## Instalation. 🚀
<br/>

### 1. Clone the repository

```bash
 git clone https://github.com/erizta/mortgage-calculator
```
<br/>

### 2. Run this command to install the dependencies.

```bash
 npm install
```
<br/>


### 3. Run this command to raise the development server.

```bash
 npm run dev
```
